# Login Audit Lab

## Scenario
Security team requested a review of recent login activity after suspected unauthorized access.

## Investigation Steps
- Identified current user using `whoami`
- Reviewed group memberships using `groups`
- Analyzed login history using `last`

## Commands Used
- whoami
- groups
- last

## Findings
Reviewed recent login sessions and verified no suspicious activity in the lab environment.

## Security Relevance
Monitoring login activity helps detect brute-force attempts, compromised accounts, and unauthorized access.

## Skills Demonstrated
- Linux user and group management awareness
- Log analysis
- Security monitoring fundamentals
- Incident documentation
